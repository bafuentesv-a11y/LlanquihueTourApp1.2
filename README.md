LlanquihueTourApp

Descripción

LlanquihueTourApp es una aplicación desarrollada en Java utilizando Programación Orientada a Objetos (POO). Su objetivo es gestionar información de tours turísticos mediante la lectura de datos desde un archivo CSV.

La aplicación permite:

* Cargar tours desde un archivo CSV.
* Mostrar todos los tours registrados.
* Filtrar tours según su precio.
* Buscar tours por nombre.
* Aplicar principios de Programación Orientada a Objetos mediante clases, objetos y organización por paquetes.

Estructura del Proyecto

Paquetes

* `app` : Contiene la clase principal Main.
* `data` : Maneja la carga de datos desde archivos.
* `model` : Contiene las clases de negocio (Tour, Guía y Operador).
* `service` : Implementa la lógica de procesamiento.
* `util` : Contiene utilidades y validaciones.

Tecnologías Utilizadas

* Java
* Maven
* NetBeans
* Archivo CSV

Funcionalidades

Listar Tours

Muestra todos los tours cargados desde el archivo CSV.

Filtrar Tours

Permite visualizar tours cuyo valor supera los $40.000.

Buscar Tour

Permite buscar un tour específico por nombre.
